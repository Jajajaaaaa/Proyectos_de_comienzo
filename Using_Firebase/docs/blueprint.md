# **App Name**: TaskTango

## Core Features:

- Task Creation: Allow users to create new tasks with descriptions and due dates.
- Task Editing: Enable users to edit existing tasks to update descriptions, due dates, or other details.
- Task Completion: Allow users to mark tasks as complete.
- Task Deletion: Enable users to delete tasks.

## Style Guidelines:

- Primary color: Light lavender (#E6E6FA) to convey calm and focus.
- Background color: Off-white (#F5F5F5) for a clean and distraction-free interface.
- Accent color: Soft black (#A9D0F5) for interactive elements like buttons and completion status.
- Body and headline font: 'PT Sans', a humanist sans-serif font with a modern and slightly warm feel, will be used for both headlines and body text.
- Use clear and simple icons to represent task actions (edit, delete, complete).
- A clean and straightforward layout to enhance usability, and put tasks front and center.
- Subtle animations when tasks are created, edited, or marked as complete.
